py4web/HTMX Demo

This application is a simple proof of concept of the py4web grid using htmx.

We have 3 pages in the app. It is build with no authentication/security just to keep it simple. Adding this functionality though py4web is trivial.

Products

A simple py4web grid allowing you to create some products.

Customers

py4web grid to maintain customers with the addition of a read-only list of orders for that customer.

Orders

The orders grid is build to have the order detail lines grid kept in an htmx target. The grid needs to know about the id of the parent record.
